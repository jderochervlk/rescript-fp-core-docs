# Reserved Keywords

> **Note**: Some of these words are reserved purely for backward compatibility.

- `and`
- `as`
- `assert`

- `begin`

- `class`
- `constraint`

- `do`
- `done`
- `downto` (also see `to`)

- `else` (used with `if`)
- `end`
- `esfun`
- `exception`
- `external`

- `false` (a `bool`, along with `true`)
- `for`
- `fun`
- `function`
- `functor`

- `if`
- `in`
- `include`
- `inherit`
- `initializer`

- `lazy`
- `let`

- `module`
- `mutable`

- `new`
- `nonrec`

- `object`
- `of`
- `open`
- `or`

- `pri`
- `pub`

- `rec`

- `sig`
- `struct`
- `switch`

- `then`
- `to`
- `true`
- `try`
- `type`

- `val`
- `virtual`

- `when`
- `while`
- `with`
